Brevísimo tutorial de R Markdown

En R

En el caso de de querer usar la interfaz normal de R, también se pueden generar los reportes, solo se tiene que cargar el paquete rmarkdown, tener instalado pandoc, y crear manualmente tu archivo ".Rmd", así ejecutas:

> install.packages("rmarkdown")
> library(rmarkdown)
> setwd(<working directory>)
> render("test.Rmd", "pdf_document")
> # o
> render("test.Rmd", "html_document")

De esta forma no se puede generar el archivo en Word, una opción sería instalar el paquete knitr, y con la función knit generar el archivo markdown (".md"), ahora puedes utilizar la función pandoc para generar el archivo, la ventaja de esto, es que te da más flexibilidad, permitiendote interactuar con las opciones que ofrece pandoc:

> install.packages("knitr")
> library(knitr)
> setwd(<working directory>)
> knit("test.Rmd")
> 
> pandoc("test.md", format = "html")  # HTML
> pandoc("test.md", format = "latex") # LaTeX/PDF
> pandoc("test.md", format = "docx")  # MS Word
> pandoc("test.md", format = "odt")   # OpenDocument
