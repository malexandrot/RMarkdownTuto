Brevísimo tutorial de R Markdown

Sintáxis básica de markdown

Toda esta información se puede encontrar en la página de Markdown, esto sería lo "necesario" para empezar, pero se pueden hacer muchas cosas más:

Encabezados:

Dependiendo de su importancia:

> # Importante
> ## Menos importante
> ### Mucho menos importante

Listas

  • Ordenadas:
> 1. Primer elemento
> 2. Segundo elmento
> 3. Tercer elemento
  • No ordenadas:
> * Elemento
> * Elemento
> * Elmento
  • Listas anidadas:
> * Primer nivel
> ** Segundo nivel

Imágenes:

> ![alt text for image](http://ejemplo.com/image.jpg)

Código:

En el caso de markdown se usa ``` código ```, pero en el caso de querer colocar código en R, se usa lo que se conoce como :

```{r opciones}
    x <- rnorm(100)
    y <- 2*x + rnorm(100)
```

Donde representa las distintas opciones que se pueden utilizar en el , donde algunas de las más usadas son:

echo = FALSE # No muestra el código
results = 'hide' # No muestra los resultados del código
warning = FALSE # Omite todos los warnings del código
eval = FALSE # No ejecuta el código

Tablas

Se pueden crear tablas sencillas de la siguiente manera:

| Media  | Varianza | Mediana | Rango  | CV    |
|--------|----------|---------|--------|-------|
| 134.03 | 1504.045 | 139.30  | 162.21 | 29.94 |

Ecuaciones

Con R markdown puedes agregar ecuaciones usando la sintáxis de :

$ecuación en una línea$
$$ecuación$$
